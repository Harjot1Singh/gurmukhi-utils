## Usage

```javascript
const { toUnicode } = require('gurmukhi-utils')

toUnicode('Koj')    // Returns ਖੋਜ
```

## Docs

Currently, there's only one function.

- `toUnicode()` - converts the ASCII representation used in GurmukhiAkhar fonts to Unicode
